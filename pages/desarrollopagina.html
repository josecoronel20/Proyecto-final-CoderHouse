Desarrollo de ésta página web

Esta página se desarrolló con el objetivo de mostrar lo aprendido en el curso de desarrollo web de CoderHouse. Ésta página simularia una página donde los usuarios podrían pedir información sobre presupuesto de un sitio web y en caso de que lo necesiten, también podrían pedir su propio logo. En este sitio se encuentran páginas tales como la página de inicio, una página con información sobre la empresa ficticia, la galeria de fotos de trabajos, y un formulario para pedir presupuestos.

En ésta sección voy a explicar que tecnologías y en que partes se utilizaron.

Bootstrap

Se empezó a utilizar lo básico de Bootstrap, se agregó el navbar responsive, el formulario, las cards y el accordion de Bootstrap, y se modoficó los colores, los margin y padding mediante clases buscando un resultado más acorde a la página. todavia falta por explorar mucho de bootstrap pero con saber aplicar los componentes y editarlos es suficiente para la entrega.

SASS

Se empezó a utilizar lo básico de Sass, variables, mixins, y extends, tambíen se comenzó a probar las clasicas anidaciones de Sass. Obviamente para la utilización de Sass se necesito hacer la instalación de el mismo. se utilizó la forma de instalación más larga con instrucciones del profesor, se editó el archivo package.json y se ignoró la carpeta node_modules para ser subido a GitHub.

GIT y GitHub

También fue necesaria la instalaion y posterior utilizaciónde GIT, se comenzó a utiizar los clasicos comandos de branchs y commits.

Después de aprender a usar lo básico de GIT se subieron todos los archivos a GitHub para también aprender a utilizarlo y además utilizarlo como sitio de correcciones para los tutores.

Logo

por último se utilizó Illustrator para crear el logo desde 0, ésto lo agrego por mi cuenta ya que poseía conocimientos básicos de Illustrator y Photoshop.
